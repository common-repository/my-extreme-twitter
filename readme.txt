== Description ==

This plugin will add an widget to use Twitter's tweet.

Plugin Features

* Twitter stream
* Header enable/Disable
* Theme selection
* You have many option how you want to use your fan page like page.
* Very Lightweight.
* Easy to use
& many More.

== Installation ==

* Upload `myextremeTwitter.zip` to the `/wp-content/plugins/` directory and extract it.
* Activate the plugin through the 'Plugins' menu in WordPress
* Use widget "My Extreme Twitter" and Put your username
* Now customize it.
